#📚 NEU Library Visitor Log System

#📌 Description
The NEU Library Visitor Log System is a web-based application designed to efficiently record, manage, and monitor library visitors within New Era University. The system streamlines the visitor logging process by allowing users to check in using either manual entry or Google authentication, ensuring that only valid institutional email accounts are accepted.
Administrators are provided with a secure dashboard where they can view visitor statistics, monitor activity, manage user access, and analyze trends through visual charts. The system improves record accuracy, reduces manual tracking, and enhances overall library management.

🚀 Features
🔐 Google Authentication (NEU email only)
📝 Visitor Check-In Form
📊 Admin Dashboard with Analytics
📈 Charts (Bar & Pie) for Visitor Insights
🔍 Search & Filter Visitor Records
🚫 Block / Unblock Users
🏫 Visitor Tracking by College
📅 Daily, Weekly, Monthly Reports
